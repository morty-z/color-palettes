# Linear

Sleek, focused, and engineering-driven. Inspired by Linear's dark interface for issue tracking, this theme features a near-black background with an indigo accent that feels fast and purposeful. It is designed for developer-facing tools, changelogs, technical writing, and any content that benefits from a refined dark mode.

- **Mode**: dark
- **Font category**: sans-serif
- **Best for**: developer tools, changelogs, technical writing, issue trackers, dark-mode dashboards
