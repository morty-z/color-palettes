# Stripe

Polished, developer-friendly, and precise. Inspired by Stripe's acclaimed documentation and marketing pages, this theme uses a cool steel-blue background with a vivid purple accent. It conveys technical sophistication while remaining approachable, making it perfect for API docs, fintech products, and developer tools.

- **Mode**: light
- **Font category**: sans-serif
- **Best for**: API documentation, fintech products, developer tools, SaaS landing pages
