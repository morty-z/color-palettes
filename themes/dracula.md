# Dracula

Dracula is a beloved dark theme originating from the developer community, featuring a distinctive dark blue-gray background with vibrant pastel accents. Each heading level uses a different color from the Dracula palette -- purple, pink, cyan, and green -- creating a colorful yet harmonious hierarchy. Ideal for developer tools, code-heavy content, and dark-mode interfaces.

- **Mode**: dark
- **Font category**: sans-serif
- **Best for**: developer tools, code documentation, dark-mode apps, terminal interfaces, tech blogs
