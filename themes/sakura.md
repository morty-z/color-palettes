# Sakura

Sakura is a soft, feminine light theme inspired by Japanese cherry blossoms, featuring gentle pinks and warm purples on a delicate blush background. It uses generous border-radius values and subtle background highlights on bold text to create a warm, inviting aesthetic. Perfect for personal blogs, lifestyle content, creative portfolios, and any project that values elegance and softness.

- **Mode**: light
- **Font category**: sans-serif
- **Best for**: personal blogs, lifestyle content, creative portfolios, journals, wedding sites
