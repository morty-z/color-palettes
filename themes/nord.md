# Nord

Nord is an arctic-inspired dark theme built on a cool, blue-gray palette that evokes Scandinavian minimalism. It uses muted frost blues and soft greens for accents, creating a calm and focused reading experience without harsh contrast. Well suited for note-taking apps, developer dashboards, and any interface that benefits from a soothing dark mode.

- **Mode**: dark
- **Font category**: sans-serif
- **Best for**: note-taking apps, developer dashboards, dark-mode interfaces, productivity tools
