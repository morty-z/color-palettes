# Bloomberg

Bold, data-dense, and terminal-inspired. Inspired by the Bloomberg Terminal aesthetic, this theme uses a pure black background with bright terminal green text and an orange secondary accent. It is built for financial dashboards, data-heavy content, real-time feeds, and any project that wants to channel the energy of a trading floor.

- **Mode**: dark
- **Font category**: monospace
- **Best for**: financial dashboards, data terminals, real-time feeds, trading interfaces, hacker aesthetics
