# Monokai

Monokai is a legendary syntax-highlighting theme originating from Sublime Text, featuring a warm dark olive background with high-contrast neon accents. Its monospace font stack and compact sizing give it a distinctly code-editor feel, while the vivid pink, green, yellow, and cyan heading hierarchy mirrors classic syntax coloring. Ideal for developer blogs, code tutorials, and technical content where a hacker aesthetic is desired.

- **Mode**: dark
- **Font category**: monospace
- **Best for**: developer blogs, code tutorials, terminal-style interfaces, hacker aesthetic, technical writing
