# color-palettes

30 pre-matched color palettes for any project. Describe the vibe, get 10 hex values that work together.

> "I want something warm" → **Sunset**: `#fdf6ee` bg, `#c0582a` accent, `#3d2c1e` text...

## What This Is

Each palette is **10 hex colors** verified to work together — backgrounds, accents, text hierarchy, borders. Inspired by real products (Notion, Linear, Stripe) and design movements (Bauhaus, Cyberpunk, Nord).

Not a design system. Just colors.

## What's in Each Palette

10 hex colors:

```
background      → page background
surface         → card/section background
primary         → main accent color
accent          → secondary accent
text            → body text
text-secondary  → subheadings, captions
text-muted      → metadata, placeholders
border          → dividers, input borders
code-bg         → code block background
blockquote-border → quote border
```

## Installation

```bash
git clone https://github.com/<your-username>/color-palettes ~/.claude/skills/color-palettes
```

## Usage

Just describe what you want:

- *"warm colors"* → Sunset, Coffee, or Claude
- *"dark and techy"* → Linear or Cyberpunk
- *"like Notion"* → Notion
- *"cute and pink"* → Sakura or Pastel

Claude outputs the 10 hex values, optionally formatted for your framework (Tailwind, CSS vars, SCSS, JS).
